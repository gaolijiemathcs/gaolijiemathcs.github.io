2.cmp函数的编写

sort函数进行排序的时候，需要提供cmp函数实现的排序规则。

以学生的例子为，可以依据排序规则，定义下列的表达式：

①如果两个学生的分数不同，则高分排在前面

②否则将名字字典序小的排在前面

此处需要对strcmp函数做一个解释：strcmp函数是string.h头文件下的用于比价两个char型数组的字典序的大小的，其中strcmp(str1, str2)当str1的字典序小于str2的时候返回一个负数，当str1的字典序等于str2的时候返回0.当str1的字典序大于str2时返回一个整数。对此处写的

理解：如果字符数组可以直接比较，那么当需要按字典序从小到大排序的就会在cmp中写类似return a.name < b.name的写法，表示a.name的字典序小于b.name的字典序。那么，当使用strcmp进行比较的时候，实际上是采用同一个思路，即使用strcmp(a.name, b.name) < 0, 表示a.name的字典序小于b.name的字典序。唯一需要注意的是strcmp的返回值不一定是-1或+1(与编译器有关)，故return strcmp(a.name, b.name) == -1是错误的。

3.排名的实现

很多排序题都会要求在排序之后计算出每个个体的排名，并且规则一般是：分数不同的排名不同，分数相同的排名相同但占用一个排位。例如有五个学生的分数分别为90/88/88/88/86，那么这五个学生的排名分别为1/2/2/2/5。
