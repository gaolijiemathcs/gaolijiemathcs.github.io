A1051题解

发表于 2019-02-26

本题模拟题需要注意的是整个的过程与理解算例。注意栈的大小是有限制的，同时队列入栈的数列大小与栈的大小不同需要区分。要判断是否是合理的数列因此加上bool flag; 当中使用current来指向当前的字母。同时要注意这里使用的是while循环这样可以有利于多次入栈后再多次出栈只要实现了!st.empty() && st.top() == arr[current] 就可以多次出栈 并且这里是一个一个的读取的 读取了一个。这里的current十分巧妙要好好体会。
