string的常见用法详解

发表于 2019-02-10

在C语言中，一般使用字符数组char str[]来存放字符串，但是使用字符数组有时会显得操作麻烦，而且容易因经验不足而产生一些错误。为了使得编程者可以更方便地对字符串进行操作，C++在STL中加入string类型，对于字符串常用的需求功能进行了封装，使得操作更加方便。

如果使用string需要加上string头文件，因此需要加上”#include“(需要注意string 与 string.h是不一样的头文件).同时加上using namespace std;

(9)find()

str.find(str2),当str2是str的子串时，返回其在str中第一次存在的位置。如果str2不是str的自串的时候，那么返回string::npos。

str.find(str2, pos)，从str的pos号位置开始匹配str2,返回值与上相同。

时间复杂度为O(nm)，其中n和m分别为str 和str2的长度。
