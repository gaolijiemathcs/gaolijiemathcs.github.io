vector的常见用法

发表于 2019-02-07

C语言中很多都需要我们自己实现，实现不合适的情况易出错，有些复杂的操作写出来十分的麻烦。C++为使用者提供了标准模板库（Standard Template Library，STL）,其中封装了很多的相当实用的容器，不需要费力的实现他们的细节直接调用函数就可以实现功能，十分的方便。初学的时候不应该被STL的名字吓到，实际上是很容易的东西，学会它对程序的简化有着十分显著的效果

我们首先学习vector的用法。

vector翻译为向量，但这里使用“变长数组”的叫法更加容易理解，也即“长度根据需要而自动改变的数组”。在考试的过程中，有时候会碰到只使用普通数组会超内存的情况， 这种情况使用vector会让问题的解决方便很多。另外vector还可以用来以邻接表的方式储存图，这对无法使用邻接矩阵的题目(结点数太多)、又害怕使用指针实现临接表的人是非常友好的，写法也相当简洁。

如果使用vector则需要添加vector的头文件，即#include。除此之外，还需要在头文件下面加上”using namespace std;”这样就可以在代码中使用vector。下面来看一些vector的常用用法。
