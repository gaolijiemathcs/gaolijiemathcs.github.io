set的常见用法详解

发表于 2019-02-08

set翻译为集合，是一个内部自动有序且不含重复元素的容器。在考试中，有可能出现需要去掉重复元素的情况，而且有可能因这些元素比较大或者类型不是int的类型而不能直接开散列表，这种情况下就可以用set来保留元素本身而不考虑它的个数。当然上面说的情况也可以通过再开一个数组进行下标和元素的对应来解决，但是set容器提供了更加直观的接口，并且加入set后可以实现自动排序，因此熟练使用set可以在做某些题时减少饲喂量。

如果要使用set，需要增加头文件set，即#include。除此之外，还需要在头文件下面再加上一个:”using namespace std;”，这样就可以在代码中使用set了。

3.set的常见用途

set最主要的作用是自动去重并且按升序排序，因此碰到需要去重但开数组较为困难的时候，可以使用set解决。

延伸：set中元素是唯一的，如果需要解决不唯一的情况需要使用multiset。另外C++11中还增加了unordered_set，利用散列表代替set内部的红黑树(一种自平衡二叉查找树)实现，使其可以用来处理只去重但不排序的需求。
